SCHEMA RULES:
- recipe_name              : string — name of the recipe inferred from transcript context
- servings_estimated       : integer — estimated number of servings the recipe produces
- amount_per_serving       : object — nutrition_total divided by servings_estimated, rounded to 1 decimal place
- ingredients              : array — one object per ingredient mentioned in the transcript
- nutrition_total          : object — sum of all ingredient nutritional values

INGREDIENT FIELD RULES:
- ingredient_name_original : string — ingredient name exactly as spoken in the transcript
- ingredient_name_en       : string — English translation of the ingredient
- quantity_value           : number — numeric quantity as spoken; use 1 if unspecified

- total_fat_g              : number — total fat in grams
- saturated_fat_g          : number — saturated fat in grams
- cholesterol_mg           : number — cholesterol in milligrams
- sodium_mg                : number — sodium in milligrams
- total_carbohydrate_g     : number — total carbohydrate in grams
- dietary_fiber_g          : number — dietary fiber in grams
- total_sugars_g           : number — total sugars in grams
- protein_g                : number — protein in grams
- vitamin_d_mcg            : number — vitamin D in micrograms
- calcium_mg               : number — calcium in milligrams
- iron_mg                  : number — iron in milligrams
- potassium_mg             : number — potassium in milligrams

UNIT CONVERSION TABLE:
- sudu besar (tablespoon)  : 15g for liquids, 12g for dry ingredients
- sudu kecil (teaspoon)    : 5g for liquids, 4g for dry ingredients
- ulas (clove)             : 3g per clove
- biji (piece/egg)         : 50g per egg, 120g per medium onion
- pinggan (plate of rice)  : 200g per plate
- ketul (piece)            : 2g for dried chili, 30g for meat piece
- cawan (cup)              : 240ml or 200g for rice
- secukup rasa (to taste)  : 2g

HANDLING RULES:
- Extract only ingredients explicitly mentioned in the transcript
- Do not hallucinate or add ingredients not present in the transcript
- If quantity is ambiguous, use the most common culinary default for that ingredient
- If unit cannot be determined, use estimated_weight_g based on context
- All numeric fields must be rounded to 1 decimal place
- nutrition_total must be the exact field-by-field sum of all ingredient values
- amount_per_serving must be nutrition_total divided by servings_estimated
- If servings cannot be determined from transcript, default to 2
- JSON must be valid and parseable — no trailing commas, no comments
