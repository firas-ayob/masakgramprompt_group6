# MasakGramPrompt

**Group:** 6
**Course:** BITP 3123 Distributed Application Development

## Project Description
This is the MasakGramPrompt project for nutritional analytics from code-switched gastronomy influencer reels using prompt engineering and large language models.

## Project Structure
- `/project` - Contains the complete Java source code and Maven configuration.
- `/transcription` - Contains all 40 transcript files used for the experiments.
